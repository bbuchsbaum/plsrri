# PLS Diverging Fill Scale

ggplot2 fill scale for diverging data.

## Usage

``` r
scale_fill_pls(..., midpoint = 0, limits = NULL)
```

## Arguments

- ...:

  Arguments passed to scale_fill_gradientn

- midpoint:

  Midpoint of diverging scale (default 0)

- limits:

  Scale limits

## Value

A ggplot2 scale object
