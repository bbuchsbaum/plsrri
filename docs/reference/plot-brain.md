# Brain Slice Plots for PLS

Functions for visualizing brain maps (saliences, BSR) using neuroim2's
plotting infrastructure. This provides thin wrappers around
[`neuroim2::plot_montage`](https://bbuchsbaum.github.io/neuroim2/reference/plot_montage.html),
[`neuroim2::plot_overlay`](https://bbuchsbaum.github.io/neuroim2/reference/plot_overlay.html),
and
[`neuroim2::plot_ortho`](https://bbuchsbaum.github.io/neuroim2/reference/plot_ortho.html).
