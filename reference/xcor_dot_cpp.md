# Fast Cross-Correlation (Dot Product)

Computes dot product between design and data matrices.

## Usage

``` r
xcor_dot_cpp(design, datamat)
```

## Arguments

- design:

  Design matrix (n_obs x n_design)

- datamat:

  Data matrix (n_obs x n_features)

## Value

Cross-product matrix (n_design x n_features)
