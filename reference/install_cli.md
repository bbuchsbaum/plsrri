# Install `plsrri` Command Wrappers

Install `plsrri` Command Wrappers

## Usage

``` r
install_cli(dest_dir = "~/.local/bin", overwrite = FALSE, commands = NULL)
```

## Arguments

- dest_dir:

  Destination directory for installed commands.

- overwrite:

  Overwrite existing command wrappers if `TRUE`.

- commands:

  Optional character vector of command names to install.

## Value

Invisibly returns the installed wrapper paths.
