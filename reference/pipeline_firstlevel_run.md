# Run First-Level Estimation

Run First-Level Estimation

## Usage

``` r
pipeline_firstlevel_run(spec, work_id = NULL)
```

## Arguments

- spec:

  Path to a YAML file or a pipeline specification list.

- work_id:

  Optional work-unit identifier from
  [`pipeline_firstlevel_plan()`](https://bbuchsbaum.github.io/plsrri/reference/pipeline_firstlevel_plan.md).

## Value

A data.frame of first-level map outputs.
