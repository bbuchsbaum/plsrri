**surface_viewer_server(id, result_rv, filters, renderer = NULL)**

Server logic:
1. **Renderer setup** (same pattern as brain_viewer_server)
   - If renderer NULL, check neurosurf availability
   - If neurosurf available, create SurfwidgetRenderer via registry
   - If not available, show fallback message
   - Store active_renderer reactive

2. **Local state**
   - geometry: Current surface geometry (reactiveVal)
   - visible_hemispheres: Which hemispheres to show
   - surface_stale: Whether surface needs re-render
   - loading: Whether currently loading

3. **Geometry change handler**
   - observeEvent(input$geometry): Update renderer geometry, mark stale, trigger re-render

4. **Hemisphere toggle handler**
   - observeEvent(input$hemispheres): Show/hide containers via shinyjs

5. **Surface rendering** (for each hemisphere)
   ```r
   output$surface_lh <- renderSurfwidget({
     req(active_renderer)
     req(result_rv())

     # Show loading
     shinyjs::removeClass("loading_spinner", "hidden")

     result <- result_rv()
     lv <- filters$lv()
     what <- filters$what() %||% "bsr"
     threshold <- filters$bsr_threshold() %||% 3.0

     # Render via abstraction
     widget <- tryCatch({
       active_renderer$render(
         result = result,
         lv = lv,
         what = what,
         threshold = threshold,
         hemisphere = "lh"
       )
     }, error = function(e) NULL)

     # Hide loading
     shinyjs::addClass("loading_spinner", "hidden")

     widget
   })
   ```

6. **WebGL resource disposal (CRITICAL for memory management)**
   Add cleanup handlers to prevent WebGL memory leaks:

   ```r
   # Track widget IDs for cleanup
   widget_ids <- reactiveValues(lh = NULL, rh = NULL)

   # Dispose WebGL resources when tab switches away
   # This is called from parent module when tab changes
   dispose_widgets <- function() {
     # Send JS message to dispose surfwidget WebGL context
     if (!is.null(widget_ids$lh)) {
       session$sendCustomMessage("dispose_surfwidget", widget_ids$lh)
     }
     if (!is.null(widget_ids$rh)) {
       session$sendCustomMessage("dispose_surfwidget", widget_ids$rh)
     }
   }

   # Register session cleanup handler
   session$onSessionEnded(function() {
     # Renderer cleanup (if it has dispose method)
     if (!is.null(active_renderer) && "dispose" %in% names(active_renderer)) {
       active_renderer$dispose()
     }
   })

   # Return dispose function for parent module to call on tab switch
   return(list(
     dispose = dispose_widgets
   ))
   ```

7. **Fallback output** (when neurosurf not available)
   - Render placeholder div with message: "Surface visualization requires neurosurf package"

8. **Colorbar info** (same as brain_viewer)
   - Use format_colorbar_label from fct_brain_viewer.R

Add data-test attributes per conventions:
